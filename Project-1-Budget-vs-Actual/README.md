
# Budget vs Actual Performance Dashboard

## Business Objective

- Monitor Budget vs Actual performance
- Identify budget variances
- Support management decision-making

## Key Insights

- Actual spending exceeded budget by 3.78%
- R&D recorded the highest budget variance (8.95%)
- Materials represented the largest expense category
